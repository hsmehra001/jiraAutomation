How It Works

  1. Choose an automation: Select either task processing or subtask creation
  2. Enter story key: Provide the Jira story key (e.g., PROJ-123)
  3. Authenticate: Use your Jira Personal Access Token if not configured
  4. Execute: Click the button and watch the automation work
